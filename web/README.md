# AI Legal Advisor Landing Page

A clean, Notion-inspired landing page for a browser extension that helps users understand Terms of Service and Privacy Policies.

## Tech Stack

- **Vite** - Build tool
- **React** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Design Principles

- Minimal, clean design inspired by Notion
- White background with soft gray text
- Subtle animations on scroll
- Max content width of 720px
- Fully responsive
